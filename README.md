# symbolicate-tool
## Tool to batch symbolicate crash files

![screenshot](https://github.com/chronicqazxc/symbolicate-tool/blob/master/.screenshot/screenshot.png)

### Install
```shell
sh install.sh
```

### Uninstall
```shell
sh uninstall.sh
```

### Usage
```shell
symbolicate-tool Xcode_name dir_of_crash_logs
```
